# Agency OS: Getting Started Guide

Welcome to Agency OS, your Enterprise-Ready Virtual Agency Platform. This guide will help you navigate your new multi-agent environment, turning high-level concepts and standard business documents into fully orchestrated, executed projects across multiple specialized domains.

## 1. Onboarding and Workspace Setup

Agency OS is built with enterprise security and multi-tenancy from day one. When you first log in, you will be placed into a Secure Workspace.

*   **Creating a Workspace:** If you haven't been invited to an existing workspace, you can create a new one. Workspaces isolate your projects, agents, and data.
*   **Role-Based Access Control (RBAC):** Invite your team members and assign them appropriate roles (e.g., Admin, Editor, Viewer) to ensure secure collaboration.
*   **Audit Logging:** Every action taken by both human users and AI agents is logged automatically. You can review the Audit Log to track who (or what agent) made specific changes.

## 2. Navigating the Interface

The Agency OS platform is designed to be as intuitive as talking to a colleague. You won't need to write a single line of code to orchestrate your virtual agency.

*   **Chat-Centric UI:** The primary way you interact with Agency OS is through natural language. Think of this as your direct line to your Virtual Product Manager.
*   **Execution Dashboard:** Once a project is initiated, the Execution Dashboard provides real-time transparency. You can monitor the progress of tasks, review outputs from specialized agents, and provide feedback to steer the project.
*   **Custom Agent Wizard:** Located in the platform settings, this no-code interface allows you to define and deploy highly specialized custom agents instantly.

## 3. Core Features: From Conversation to Creation

Agency OS operates on a unique "Conversation to Creation" model. Here is how you can use the primary features to build your next project.

### 3.1. Initiating a Project

You have two primary ways to start a project:

1.  **Conversational Project Creation:** Simply start typing in the main chat interface. Describe what you want to build (e.g., "I want to build a mobile app for dog walkers").
2.  **Document Ingestion:** Upload existing unstructured documents such as PDFs or Markdown files (e.g., an existing Pitch Deck or a rough brief). The platform will automatically extract the core requirements.

### 3.2. Auto-Scoping and the PRD

Once you provide the initial idea, the **Auto-Scoping Engine** takes over. 
*   The system acts as a senior Product Manager, asking you clarifying questions to ensure it fully understands the project.
*   It then automatically generates a comprehensive **Product Requirements Document (PRD)**.
*   You will have the opportunity to review and approve the PRD before execution begins.

### 3.3. The Nexus Pipeline

After the PRD is approved, Agency OS translates the project into an actionable Execution Pipeline.
*   **Intelligent Routing:** The **Nexus Pipeline** automatically breaks the project down into sub-tasks and routes them to the appropriate specialized agents (e.g., a UX Designer agent for mockups, a Frontend Developer agent for code, a Copywriter agent for marketing text).
*   **Transparent Execution:** You can watch the agents collaborate and execute their tasks on the Execution Dashboard. You maintain full oversight and can step in to provide feedback at any point.

### 3.4. Creating Custom Agents

Not limited to our default roster? Build your own.

*   Open the **Custom Agent Wizard**.
*   Define the agent's role, expertise, and specific instructions using plain English (e.g., "Salesforce Integration Expert" or "Brand Voice Copywriter").
*   The wizard securely deploys this agent into your workspace, making it immediately available to join your Nexus Pipelines without requiring any platform redeployment.

## 4. Next Steps

Now that you understand the basics, try initiating your first project! Start a conversation or upload a document, and watch your Virtual Agency come to life.
